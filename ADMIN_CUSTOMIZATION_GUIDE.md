# Guia de Personalização do Cardápio Digital Universal

## Acesso à Área de Personalização

1. Faça login na área administrativa (admin.html)
2. Clique na aba "Mesas & QR Codes"
3. Isso o levará para a página de gerenciamento de QR codes (qr-management.html)
4. Clique na aba "Configurações" para acessar as opções de personalização

## Opções de Personalização

### Cores do Estabelecimento
- **Cor Primária**: Define a cor principal do cabeçalho e botões
- **Cor Secundária**: Define a cor secundária usada em gradientes

### Logo e Imagem de Fundo
- **Logo do Estabelecimento**: Faça upload da logo do seu negócio
- **Imagem de Fundo**: Faça upload de uma imagem para usar como fundo da página

### Informações do Estabelecimento
- Nome do Estabelecimento
- Slogan/Subtítulo
- Endereço completo
- Telefone de contato
- Horário de funcionamento

### Redes Sociais
- Links para Facebook, Instagram e WhatsApp

## Geração de QR Code

O sistema gera automaticamente um QR code único para todo o seu estabelecimento:

1. Este é o **único QR code** necessário para todo o restaurante
2. Imprima cópias deste mesmo QR code e posicione em todas as mesas
3. Os clientes escaneiam este QR code para acessar o cardápio digital

## URL Personalizada

Seu estabelecimento recebe uma URL única:
- Formato: `https://cardapiodigital.com.br/nome-do-seu-estabelecimento`
- Esta URL pode ser copiada e compartilhada
- Leva diretamente ao cardápio do seu estabelecimento

## Como Salvar as Personalizações

1. Preencha todas as informações desejadas
2. Faça upload da logo e imagem de fundo, se desejar
3. Clique no botão "Salvar Personalizações"
4. Use o botão "Visualizar" para ver como as cores ficarão

## Visualização ao Cliente

Após salvar as personalizações:
- Os clientes verão as cores escolhidas no cabeçalho
- A logo aparecerá no topo da página
- A imagem de fundo será exibida ao fundo
- Todas as informações do estabelecimento serão mostradas

## Dicas Importantes

1. **QR Code Único**: Use apenas um QR code para todo o estabelecimento, não um por mesa
2. **Imagens**: Use imagens de boa qualidade nos tamanhos recomendados
3. **Cores**: Escolha cores que combinem com a identidade visual do seu estabelecimento
4. **Informações Atualizadas**: Mantenha as informações sempre atualizadas

## Problemas Comuns

Se as personalizações não aparecerem para os clientes:
1. Verifique se clicou em "Salvar Personalizações"
2. Peça a um cliente para testar o QR code em modo anônimo
3. Limpe o cache do navegador se necessário
