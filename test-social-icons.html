Teste de Ícones Animados de Redes Sociais

Ícones Animados

Os ícones abaixo devem estar girando continuamente em 360 graus na horizontal:

Efeito: Ao passar o mouse sobre os ícones, a animação pausa e eles aumentam levemente.

Código CSS Utilizado

/* Animação de rotação 360 graus */
@keyframes rotate360 {
    from {
        transform: rotateY(0deg);
    }
    to {
        transform: rotateY(360deg);
    }
}

.animated-icon {
    display: inline-block;
    animation: rotate360 2s linear infinite;
    transition: transform 0.3s ease;
}

.animated-icon:hover {
    animation-play-state: paused;
    transform: scale(1.2);
}
