🧪 Teste - Seção de Pagamento Delivery

Instruções

  1. Abra o arquivo delivery.html no navegador
  2. Adicione itens ao carrinho
  3. Clique em "Finalizar Pedido" ou "Ir para Pagamento"
  4. A seção de pagamento deve aparecer com as opções PIX e Cartão
  5. Selecione uma forma de pagamento
  6. O pedido deve ser confirmado

Elementos para Verificar

  • ✅ A seção de pagamento (paymentSection) existe no HTML
  • ✅ O botão "Ir para Pagamento" está visível no carrinho
  • ✅ As opções PIX e Cartão estão visíveis
  • ✅ Ao clicar em uma opção, ela fica destacada (azul)
  • ✅ O botão "Confirmar Pedido" aparece após selecionar pagamento
  • ✅ Validação ocorre se tentar finalizar sem pagamento
